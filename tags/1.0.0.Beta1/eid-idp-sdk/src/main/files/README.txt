README for eID Identity Provider SDK
====================================

=== 1. Overview

This ZIP contains the eID Identity Provider SDK.


=== 2. Installation

Just unpack the ZIP.


=== 3. Manual

The eID Identity Provider Developers Guide can be found in eid-idp-devel-guide.pdf.


=== 4. API Documentation

The API documentation can be found in the 'javadoc' directory.
